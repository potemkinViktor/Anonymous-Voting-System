# Проект Анонимная система голосования на блокчейн с использованием `zero knowledge proof`

## Данный репозиторий состоит из нескольких составляющих

1. `Contrcts` - написан на языке Solidity и состоит из трех контрактов `Factory.sol`, `Verifier.sol`, `Vote.sol`.

2. `Scripts` - написан на языке Typescript и состоит из скрипта деплоя контрактов в блокчейн и ключей шифрования для использования технологии zkSNARKs.

3. `Test` - написан на языке Typescript и состоит из трех файлов тестирования контрактов из пункта 1. Файлы имеют соотвествующие названия

4. `Public` - написан на языке Typescript и состоит из программного кода страниц frontend.

5. `Src` - написан на языке Typescript и состоит из компонентов frontend части, которые написаны оптимизироавнно для многоратного использования в различных частях, это делается с целью оптимизации работы сайта и для чистоты кода.

6. `Static` - данные которые используются для функционирования всей системы.

7. Остальные файлы - это файлы настройки среды: `hardhat.config` - конфиг настройки локального блокчейн (тестирование, компиляция и деплой контрактов).
